# Theme Tokens

The wallet apps use the TrustCare semantic token names found in the current backend repository's `client/src/index.css`.

Web keeps CSS variables and OKLCH-compatible intent. Mobile uses hex approximations exported from `packages/design-tokens`.

Key token groups:

- background, foreground, card, popover
- primary, secondary, muted, accent, destructive
- border, input, ring
- sidebar and chart colors
- radius scale
- document-type gradient families
- trust badges
